# BountyLens Publish-Ready Pack

This folder is a manual upload companion pack for publishing BountyLens to Gumroad or Lemon Squeezy.

Use it to copy product fields, choose screenshots, set the launch price, configure the post-purchase email, and answer common support questions.

No platform login, product upload, payout setup, KYC, tax, payment, withdrawal, cookie, keychain, password manager, or credential handling is performed by these files.

## Recommended Launch Setup

- Product: BountyLens Personal
- Price: $19
- File to upload: `dist/bountylens-0.1.0.zip`
- Main image: `screenshots/png/01-extension-panel.png`
- License keys: off for v0.1

## Files

- `gumroad-upload-fields.md` - Copy-ready Gumroad fields.
- `lemon-squeezy-upload-fields.md` - Copy-ready Lemon Squeezy fields.
- `image-upload-order.md` - Screenshot upload order and captions.
- `final-price-recommendation.md` - Final launch pricing rationale.
- `launch-checklist.md` - Manual Gumroad and Lemon Squeezy checklist.
- `post-purchase-email.md` - Buyer email template.
- `support-response-templates.md` - Common support replies.
- `product-metadata.json` - Structured product metadata.

## Manual Boundary

The seller must manually create the product, upload the ZIP, upload screenshots, paste copy, set policies, configure payout/tax/KYC, preview, and publish.
