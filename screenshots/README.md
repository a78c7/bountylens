# BountyLens Screenshot Assets

These files are local mock screenshots for Gumroad, Lemon Squeezy, Product Hunt, X, LinkedIn, and Reddit launch materials.

Open any HTML file in a browser and take a screenshot at 1280 x 720 or 1600 x 900.

Recommended upload order:

1. `screenshot-01-extension-panel.html`
2. `screenshot-03-cli-report.html`
3. `screenshot-04-prompt-pack.html`
4. `screenshot-06-workflow.html`
5. `screenshot-02-popup.html`
6. `screenshot-05-landing-page.html`

Primary product image:

```text
screenshots/screenshot-01-extension-panel.html
```

If PNG files exist in `screenshots/png/`, use them directly. If not, use the HTML files as browser-rendered screenshot sources.
