# Code Of Conduct

## Our Pledge

We want BountyLens to be a respectful, practical, and safety-conscious open source project.

Contributors are expected to:

- Be respectful.
- Assume good intent.
- Stay focused on the technical issue.
- Avoid harassment, insults, or discriminatory language.
- Respect safety boundaries around credentials, payments, KYC, bounty claims, and automation.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment or abusive language.
- Publishing private information.
- Encouraging credential theft, payment abuse, KYC bypass, or exploit instructions.
- Pressuring maintainers to add unsafe automation.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block contributors who violate this code of conduct.
