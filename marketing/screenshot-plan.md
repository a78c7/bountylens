## Manual Screenshot Steps

1. Open a screenshot HTML file in Chrome or another browser.
2. Set the browser window to 1280 x 720 or 1600 x 900.
3. Hide browser bookmarks or sidebar UI if needed.
4. Take a screenshot.
5. Upload the image to Gumroad or Lemon Squeezy.
